 MyRent — Платформа для размещения объявлений по аренде жилья

Данный учебный проект представляет собой веб-приложение на базе Flask, предназначенное для публикации и просмотра объявлений о недвижимости.

 Реализованный функционал
Интерфейс на Bootstrap**: Использование компонентов Bootstrap 5 для создания адаптивной верстки (навигация, карточки товаров, формы).
Загрузка и хранение файлов**: Реализован механизм загрузки изображений объектов через форму с сохранением в файловую систему сервера.
Постоянное хранение данных**: Все добавленные объявления сохраняются в файл формата CSV, что обеспечивает сохранность данных после перезагрузки приложения.
Динамические шаблоны**: Использование встроенного шаблонизатора Jinja2 для генерации страниц на основе данных.

 Стек технологий
Язык программирования**: Python 3.x
Фреймворк**: Flask
Фронтенд**: HTML, CSS, Bootstrap 5
Хранение данных**: CSV (текстовая БД), статические файлы (изображения)

 Инструкция по запуску
1. Установите необходимые зависимости:
   pip install -r requirements.txt

2. Запустите сервер приложения:
   python app.py

3. Перейдите по адресу:
   http://127.0.0.1:5000
